## My Projects

### Asiaporeair: Integrated Airline Operations Ecosystem
 
[![GitHub Repo](https://img.shields.io/badge/GitHub%20Repo-100000?style=flat&logo=github&logoColor=white)](https://github.com/AbdelrahmanAhmad99/Asiaporeair) 
[![Live Documentation](https://img.shields.io/badge/Live%20Documentation-1976D2?style=flat&logo=markdown&logoColor=white)](https://abdelrahmanahmad99.github.io/Asiaporeair/)

**Project Highlights:**

- Engineered a complex, enterprise-grade backend system to simulate a major international airline's operational environment, utilizing **Clean Architecture** principles to guarantee **high scalability**, maintainability, and domain independence. The system is structured into two specialized, integrated APIs:
          
- **Customer Booking Engine System (CBES):** (*Public-Facing API*)
  - Orchestrated the entire **Booking Lifecycle**, including dynamic flight search, seat selection, and complex itinerary finalization.
  - Integrated the **Stripe API** for secure, compliant payment processing and enabled automated Refund Management workflows.
  - Developed a **Check-In Simulation Module** that updates ticket status and generates boarding passes based on pre-flight time windows (Simulated Process).
  - Implemented robust **Identity** and **JWT** authentication, enforcing **Role-Based Access Control (RBAC)** to secure all public and administrative endpoints.
  
- **Airport Operations Management System (AOMS):** (*Secure Admin API*)
  - Engineered the **Flight Scheduling Module**, enabling administrators to create recurring flight schedules and assign specific Crew Members to flight instances.
  - Managed critical system states by implementing high-level override capabilities, such as Force Status Change for tickets (e.g., Cancelled, Offloaded).
  - Developed endpoints for comprehensive **Operational Data Retrieval** (e.g., flight manifests, crew rosters, system logs).
  - Implemented **Aircraft Asset Management** for tracking fleet details and ensuring valid aircraft are assigned to routes.

**Technologies Used:**
`ASP.NET Core Web API` `C#` `Clean Architecture` `EF Core` `SQL Server` `JWT/Identity (RBAC)` `Stripe API` `RESTful` `AutoMapper` `Swagger (OpenAPI)` `MkDocs/GitHub Pages`
